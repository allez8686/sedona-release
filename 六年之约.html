🔐 配置云同步
连接 Supabase 数据库，让您的六年之约数据在多个设备间同步。只需 2 分钟完成设置。
1
创建 Supabase 项目
打开 supabase.com，注册/登录后点击 "New Project"，创建免费项目。
2
创建数据库表
在 Supabase 左侧菜单进入 SQL Editor，粘贴以下 SQL 并运行：
-- 打卡记录表 CREATE TABLE IF NOT EXISTS entries ( id BIGINT GENERATED ALWAYS AS IDENTITY PRIMARY KEY, user_id UUID REFERENCES auth.users NOT NULL, date DATE NOT NULL, mood TEXT, happy TEXT, challenge TEXT, thoughts TEXT, savings INTEGER DEFAULT 0, exercised BOOLEAN DEFAULT false, learned BOOLEAN DEFAULT false, created_at TIMESTAMPTZ DEFAULT NOW(), UNIQUE(user_id, date) ); -- 用户设置表 CREATE TABLE IF NOT EXISTS user_settings ( user_id UUID REFERENCES auth.users PRIMARY KEY, target_amount INTEGER DEFAULT 1000000 ); -- 行级安全策略 ALTER TABLE entries ENABLE ROW LEVEL SECURITY; ALTER TABLE user_settings ENABLE ROW LEVEL SECURITY; CREATE POLICY "Users manage own entries" ON entries FOR ALL USING (auth.uid() = user_id) WITH CHECK (auth.uid() = user_id); CREATE POLICY "Users manage own settings" ON user_settings FOR ALL USING (auth.uid() = user_id) WITH CHECK (auth.uid() = user_id);
3
填入连接信息
在 Supabase 左侧菜单 Settings → API，复制 Project URL 和 anon public key。
